# 📘 Treinamento Swift - POO

## 🧱 Programação Orientada a Objetos - Lista 01

Este módulo tem como objetivo praticar conceitos de Programação Orientada a Objetos (POO) utilizando Swift.

---

## 📂 Conteúdo

* 🧍 Classes e Objetos
* 🏗️ Construtores
* ⚡ Métodos
* 🔁 Sobrecarga
* 🧬 Structs

---

## 🧠 Exercícios

### 🧍 Classe Aluno

Crie uma classe `Aluno` com:

* 🔹 Nome
* 🔹 Nota 1
* 🔹 Nota 2

Funcionalidades:

* ⚡ Calcular média
* ⚡ Retornar situação (Aprovado/Reprovado)

📌 Exibir: nome, média e situação

---

### 🛒 Classe Produto

Crie uma classe `Produto` com:

* 🔹 Nome
* 🔹 Valor

Funcionalidade:

* ⚡ Aplicar desconto caso valor ≥ R$1.000,00

📌 Exibir: nome do produto e valor final

---

### 💰 Classe Descontos

Crie uma classe `Descontos` com:

* 🔹 Salário Bruto
* 🔹 Faltas

Funcionalidades:

* ⚡ Desconto de Vale Transporte (6%)
* ⚡ Desconto de IR:

| Faixa Salarial       | Desconto |
| -------------------- | -------- |
| Até R$1.560,00       | 0%       |
| Até R$3.120,00       | 4%       |
| Até R$6.240,00       | 7%       |
| Até R$12.480,00      | 12%      |
| Acima de R$12.480,00 | 15%      |

* ⚡ Bonificação:

  * +R$400,00 se não houver faltas

📌 Exibir:

* Descontos
* Bonificações
* Salário líquido

---

### 📊 Classe Media (Funções Estáticas)

Crie uma classe `Media` com métodos estáticos:

* ⚡ Média de 2 notas
* ⚡ Média de 3 notas
* ⚡ Média de 4 notas

📌 Retornar: média e situação

---

### 👤 Classe Cliente (Sobrecarga de Construtor)

Crie uma classe `Cliente` com:

* 🔹 Nome
* 🔹 Idade
* 🔹 Cidade

Construtores:

* Sem parâmetro → `Bom dia!`
* Nome → `Bom dia Ricardo!`
* Nome + idade → `Bom dia Júlia, você tem 32 anos.`
* Nome + idade + cidade → `Bom dia Rebeca, você tem 28 anos e mora em São Paulo.`

---

### 🐾 Struct Animal

Crie uma `struct Animal` com:

* 🔹 TipoAnimal
* 🔹 Peso

Atividades:

* Criar um objeto
* Criar uma cópia
* Alterar valores

📌 Exemplo esperado:

* Animal1 → Leão, 150
* Animal2 → Urso, 400
